## 📌 Overview

This project is part of the **DevelopersHub Data Science Internship - Task 1**.  
The goal is to perform exploratory data analysis (EDA) on the classic **Iris dataset** and generate three key visualizations using Python libraries.

## 📊 Visualizations

| Plot Type | Description |
|-----------|-------------|
| **Scatter Plot** | Sepal Length vs Sepal Width, colored by species |
| **Histogram** | Distribution of all four numerical features |
| **Box Plot** | Feature spread and outlier detection per species |

## 🛠️ Technologies Used

- **Python 3.8+**
- **Pandas** – Data manipulation
- **Matplotlib** – Core plotting
- **Seaborn** – Statistical visualizations
